# Backend

## Commands

`npm run serve`
Run the server at localhost:3000

`npm run lint`
Checks if there are lint errors

`npm run lint:fix`
Tries to fix lint errors. This command will format the code according to prettier too

`npm run format`
Formats the code according to prettier
